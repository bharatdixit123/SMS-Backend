# Student Management System

## 📌 Project Description
This is a Student Management System web application built using Spring Boot and Thymeleaf.
It allows users to manage student records with basic CRUD operations.

---

## 🚀 Features
- Add new student
- View all students
- Update student details
- Delete student records
- Server-side rendering using Thymeleaf
- MySQL database integration

---

## 🛠️ Technologies Used
- Java
- Spring Boot
- Spring MVC
- Spring Data JPA
- Thymeleaf
- MySQL
- Maven

---

## ⚙️ How to Run the Project

### 1. Clone the repository
git clone https://github.com/bharatdixit123/SMS-Backend.git

### 2. Open project in IDE (IntelliJ / Eclipse / VS Code)

### 3. Configure database in application.properties
spring.datasource.url=jdbc:mysql://localhost:3306/student
spring.datasource.username=root
spring.datasource.password=your_password

### 4. Run the application
Run main class: StudentManagementSystemApplication

### 5. Open in browser
http://localhost:8080
